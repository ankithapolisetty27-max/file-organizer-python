# File Organizer Project

## Description
This is a simple Python project that automatically organizes files in a folder into different categories such as Images, Documents, Videos, and Others.

The program scans a selected folder and moves files into their respective folders based on file type.

## Features
- Automatically organizes files
- Creates folders if they do not exist
- Supports multiple file types
- Easy to run using Python

## Technologies Used
- Python
- OS Module
- Shutil Module
